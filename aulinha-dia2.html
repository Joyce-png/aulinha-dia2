Como commitar e criar branches

  1. Criar repositório local, depois o arquivo via vscode
  2. Precisamos adicionar o .git no repositório: comando git init
  3. Para adicionar todas as alterações de todos os arquivos usa-se o comando "git add .", para adicionar as alterações de um arquivo específico, acrescenta o nome do arquivo. Exemplo: git add outro-arquivo.html
  4. Após adicionar (git add), pode ser geito o primeiro commit na branch principal, usando o comando "git commit" com a mensagem do commit. O comando fica: git commit -m "primeiro commit"
  5. Depois do primeiro commit, após realizadas as alterações necessárias no projeto, já pode-se transferir as alterações para o repositório remoto (GitHub). Lembrando que antes disso é necessário criar o repositório remoto no GitHub. Após a criação do repositório remoto, o próprio GitHub apresenta os próximos comandos a serem usados.
